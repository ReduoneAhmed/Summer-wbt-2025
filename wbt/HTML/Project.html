Projects

Over the course of my studies,I’ve worked on several projects
using different programming languages and tools.These
experiences have helped me build a strong foundation
in software development and problem-solving. Some of
the key areas I’ve worked on include:

E-Commerce System – Java.

Built a complete e-commerce system using Java where users could browse products, add items to the cart, and place orders. The project included login/registration features,product management, and order tracking. I used object-oriented designprinciples and Java Swing for the user interface, along with file handling for data storage.

Hospital Management System – C#.

Developed a hospital management application using C# and Windows Forms. The system handled patient records, doctor appointments, billing, and admin management. I used SQL Server for the backend database and implemented search, update, and reporting functionalities. This project gave me a solid understanding of C# event-driven programming and database integration.

Plane Bombing Game – C++ (OpenGL).

Created a 2D animated game where a plane drops bombs on buildings, using C++ and OpenGL. The game featured smooth animation, keyboard/mouse controls, and real-time interactions. It included elements like a moving background, buildings, sun, clouds, and collision effects. This project helped me strengthen my understanding of computer graphics, transformations, and rendering techniques.
